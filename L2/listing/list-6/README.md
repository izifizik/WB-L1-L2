Что выведет программа? Объяснить вывод программы.
```
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```
Ответ:
```
type Slice struct {
	Data unsafe.Pointer
	Len  int
	Cap  int
}
Data - указатель на массив => если менять массив то можно не принимать слайс в return

append - функция "добавления" элемента в массив 
Под капотом он создает новый массив на определенное колличество элементов
Единственное что не знаю имеет ли этот массив тот же адрес в памяти если мы пасширяем его Len в пределах Cap
Если расширять Cap то адрес скорее всего изменится => любые изменения нового массива не будут зафиксированны по указателю
```
